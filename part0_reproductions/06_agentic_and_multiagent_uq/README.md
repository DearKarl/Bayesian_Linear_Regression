# Route 6: Agentic and Multi-Agent Uncertainty

This route moves from answer-level uncertainty to uncertainty over claims,
turns, tool calls, plans, and complete trajectories. It also studies correlated
agent errors: agreement between agents is not independent evidence when agents
share a base model, retrieval source, or prompting strategy.

Planned work will evaluate trajectory-level proper scores and cost-sensitive
policies for answering, clarifying, retrieving, verifying, regenerating,
consulting another agent, and escalating to human review.

**Status:** literature protocol defined; no experiment has yet been claimed as
reproduced.
