# Route 4: RAG and Heterogeneous Evidence

This route separates factuality from faithfulness and studies how retrieval
quality, model uncertainty, verifier scores, and source agreement should be
combined. Representative 2026 systems include FRANQ and MIRAGE.

The long-term objective is a measurement model in which each evidence source
has an estimable reliability and correlated sources are not treated as
independent confirmation.

**Status:** literature protocol defined; no experiment has yet been claimed as
reproduced.
