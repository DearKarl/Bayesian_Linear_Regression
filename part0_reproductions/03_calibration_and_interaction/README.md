# Route 3: Calibration and Interaction

This route studies verbalized confidence, post-hoc calibration,
calibration-aware training, clarification, and multi-turn calibration. The
central question is whether an uncertainty estimate remains probabilistically
meaningful after interaction changes the context or response distribution.

Planned evaluations will include proper scoring rules, calibration curves,
subgroup calibration, and temporal calibration across dialogue turns.

**Status:** literature protocol defined; no experiment has yet been claimed as
reproduced.
